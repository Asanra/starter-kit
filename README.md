## STARTER-KIT PROJECT
This is a starter kit to help me to build a website

***

#### Features
- **Gulp tasks**
    - HTML, CSS, JS and Images optimization
    - Autoprefixer for CSS
    - Cross-device Synchronization with Browsersync
    - Live Browser Reloading
    - Convert TTF to WOFF
- **RESET CSS**
- **HTML5 Shiv**
- **Baseline Grid**
- **CSS Grid System**
